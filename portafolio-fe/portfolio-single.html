Social Pet

En este proyecto el reto fue construir una Red Social, Single-Page Application (SPA), responsiva en la que podamos escribir, leer, actualizar y eliminar datos. para realizar este reto yo cree una red social enfocada en dueños de mascotas donde los usuarios podrán interactuar, intercambiar tips, publicar un post con temas referidos a sus mascotas. Otros usuarios pueden comentar, dar like, editar su comentario al igual que algún post realizado por ellos y eliminar alguna publicación realizada cuando quieran

Detallles del proyecto
fecha: dec 12, 2017 Herramientas: HTML5, CSS3, JS y Firebase. Repositorio: www.github/social-network.com ver demo

LatinFemiData

En este proyecto el reto fue construir una web para visualizar un conjuntos(set) de datos que se adecue a las necesidades de mi usuario. para trabajar en este proyecto de parejas realizamos una investigación previa para entender quién era el usuario, que necesitaba saber o ver exactamente habiendo hecho esta previa investigación y ya entendiendo las necesidades del usuario, construimos la interfaz de LatinFemiData una página web que permite visualizar, filtrar y ordenar la data también mostrando algunos cálculos agregados como el max, min y promedio mostrando así información más relevante para el usuario para desarrollar esta aplicación usamos HTML5, CSS3, Bootstrap 4 y JavaScript. Fecha: jan 30, 2017 Herramientas: HTML5, CSS3, Bootstrap4 Repositorio: www.github/data-lovers.com ver demo

Segurity Data

En este proyecto el reto fue crear mi primera aplicación web en la que el usuario pueda cifrar y descifrar un texto, indicando un desplazamiento específico de caracteres(offset). la temática era libre así que yo desarrolle una aplicación en la que el usuario puede encriptar sus datos personales y bancarios para poder compartirlo de manera segura, los datos podrán ser desencriptados de la misma manera con el código que fue encriptado(offset). Para desarrollar esta aplicación utilice HTML5, CCS3 y JavaScript.
